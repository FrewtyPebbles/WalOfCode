<h2>About the Project:</h2>
<p></p>
	<p style = "text-indent:40px"><i>noodlePy</i> is an elaborate joke language made for fun over the course of 3-5 days in Python.  It is a language that is based around the theme of pasta.</p>
<p></p>

<h2>Noteworthy Features:</h2>
<p></p>
<ul>
  <li>Working scope and conditions.</li>
  <li>Source files are interpreted using character and word streams making indentation and newlines optional.</li>
  <li>Variable and array/list system with instantiation and assignment.</li>
  <li>A parsing system for keywords and built in methods.</li>
  <li>Working functions with recursion.</li>
</ul>
<p></p>
<hr />
<h2>Documentation:</h2>
<p></p>
<p style = "text-indent:40px">To execute a <b>.pasta</b> script, run the interpeter (noodle.exe) and enter in the name of your source file.</p>
<p></p>
 <p style = "text-indent:40px; color:#ffa8b7"><b>IMPORTANT: All of your noodle code that executes outside of functions must
 be placed within a PLAIN SCOPE (<i>? your main code ?</i>) 
 that has NO statement AFTER your function definitions.  Variable instantiations must go at the verry top 
 of your script.  Refer to the <i>example.pasta</i> source file in the project download if you are confused.</b></p>
